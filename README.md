# visor
VISOR: Logic-Based Motorcycle Weather Dashboard  A rider-focused web app that translates raw weather data into a 0-100% Ride Score. Features a color-coded 24h Commuter Timeline, smart "Wet Road" persistence logic, and dynamic day/night backgrounds. Built with Vanilla JS &amp; Open-Meteo API for instant, glanceable decision-making on mobile.
